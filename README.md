# PURSE-ACCOUNTING-MOBILE

---

[![Typing SVG](https://readme-typing-svg.herokuapp.com?size=24&lines=Purse+in+your+pocket)](https://git.io/typing-svg)

---

The PURSE-ACCOUNTING-MOBILE is Android and IOS interafece for porse-accounting.ru that allows the user to calculate the budget for the day, keep track of income and expenses, save records in a personal account.

---

CI pipeline - [GitHub](https://github.com/RolePlayGames/purse-accounting-mobile/actions)

## Technology stack

- .NET 9 - framework
- MAUI - application framework
- xUnit - testing library
- Appium - mobile testing library

## Development

### API solution linting

To lint API solution you need to open command line in /WebAPI folder and run:
```sh
dotnet format --severity warn --verify-no-changes
```
